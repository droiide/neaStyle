Code

<div class="grid"> tags </div>

You must indicate how many columns you are going to have.

The # sign is changed for the number of columns, these must be from 1 to 12

<div class="grid columns-#"> tags </div>

For this to be responsive you must indicate xl for extra large screen md for medium screens and sm for smaller screens

<div class="grid columns-# md-columns-# sm-columns-# xl-columns-6"> tags </div>


Example

<div class="grid columns-4 md-columns-2 sm-columns-1 xl-columns-6"> <div class="card"> Text or image </div> </div>
